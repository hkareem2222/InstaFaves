InstaFaves
======
**InstaFaves** is an iOS app that searches for Instagram pics with the tag you provide. You can then save and store these pics on your phone to share later.

## Technology
* Map Kit Framework
* Instagram API
* Custom Delegation
* UICollectionViewCell
* NSUserDefaults
* Saving to plist
